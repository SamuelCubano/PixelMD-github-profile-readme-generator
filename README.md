## ✨ ¿Qué es PixelMD?

**PixelMD** es una herramienta web construida con **Angular 21** que permite a desarrolladores generar perfiles de GitHub impactantes sin tocar una sola línea de código. Olvídate de los archivos `.md` aburridos; diseña tu presentación con un toque gamer y minimalista.

### 🕹️ Características Principales
*   **Interfaz Pixel-Art:** Una experiencia de usuario nostálgica pero moderna.
*   **Previsualización en Tiempo Real:** Mira cómo queda tu perfil mientras lo editas.
*   **Estilo Glassmorphism:** Componentes translúcidos y elegantes.
*   **Optimizado para Devs:** Exportación limpia y lista para copiar/pegar.

## 🚀 Instalación y Uso

Si quieres correr PixelMD localmente o contribuir al código:

1.  **Clona el tesoro:**
    ```bash
    git clone [https://github.com/SamuelCubano/PixelMD-github-profile-readme-generator.git](https://github.com/SamuelCubano/PixelMD-github-profile-readme-generator.git)
    ```
2.  **Instala las dependencias:**
    ```bash
    npm install
    ```
3.  **Inicia la aventura:**
    ```bash
    ng serve
    ```
    Ve a `http://localhost:4200/` y ¡empieza a crear!
